# 3.5.3 :root 使用

```
:root {
    --blue: #007bff;
    --indigo: #6610f2;
    --purple: #6f42c1;
    --pink: #e83e8c;
    --red: #dc3545;
    --orange: #fd7e14;
    --yellow: #ffc107;
    }
    
    
 p {
   color: var(--blue)
 }
```
