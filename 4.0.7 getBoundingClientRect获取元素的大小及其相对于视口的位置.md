# getBoundingClientRect


>返回元素的大小及其相对于视口的位置。


```
document.getElementById('round').getBoundingClientRect();

// 返回 Obj 
DOMRect {x: 435.1875, y: 103.578125, width: 720.8125, height: 27, top: 103.578125, …}
bottom: 130.578125
height: 27
left: 435.1875
right: 1156
top: 103.578125
width: 720.8125
x: 435.1875
y: 103.578125
```
